DTx SH4ALL: Therapeutics Adherence Module (iOS)

Overview

This project aims to provide the Therapeutic Adherence module from DTx (SaMD framework), implemented as an iOS binary framework (XCFramework).

The DTx SaMD framework consists of a series of components and building blocks that come together as part of a medical system. Medication Management will be inserted in the context of the Therapeutics Adherence building block / module, to manage medication prescription administrations. Expected functionalities are:

  • Manage medication prescription data
  • Expose medication prescription functionalities as services
  • Define and implement the platform supporting services
  • Collect data from medication device
  • Connect app to the medication device

Dependency management + distribution

We will emphasize the usage of Swift Package Manager over CocoaPods:

  • Improved maintainability
  • First-party solution, available for all developers out-of-the-box with Xcode 11+
  • Less development environment configuration and troubleshooting
  • Cocoapods library is bigger, but most active libraries have migrated already

This will be valid for both managing DTxTherapeuticAdherence module dependencies, as well as its distribution. Cocoapods will still be an option for distribution – for legacy reasons –, but not recommended.

Binary frameworks vs. Swift Packages

With Swift Package Manager, Swift Packages were also introduced as an alternative to Binary Frameworks (XCFramework). Both solutions are valid for the purpose of the project, with pros and cons for each one, therefore both will be implemented as a learning exercise. For the Swift Package implementation check dtx-sh4all-ios-therapeuticadherence-package.

Installation

Swift Package Manager

WIP: Update code repository URL(s) to Gitlab before migration.

  1. Open the project to which you are going to add DTxTherapeuticAdherence

  2. Open Xcode’s Project Editor to add the dependency (Select Project > Swift Packages and [ + ])

  3. In the Choose Package Repository dialog, enter the appropriate DTxTherapeuticAdherence, [ Next ] to continue

    https://github.com/ridias-altran/dtx-sh4all-ios-therapeuticadherence-framework.git

  4. Enter the latest Version tag or specify the desired one, [ Next ] to continue

  5. [ Finish ] to close the Choose Package Repository dialog.

end

  3. Run pod install to install dependencies and (re)generate .xcworkspace
